# React Todo App
This is a simple React application built using vite.
## features 
- Displays a list of todos
- Uses map() to render items
- Clean and simplecomponent structure

## Instalation
1. Clone the reository:
git clone https://github.com/mholechissa/React-26-2

2. Navigate into the roject:
cd React-26-2

3. Install dependencies:
npm install

4. Run the development server:
npm run Dev
